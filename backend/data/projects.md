## GUITestBench：面向移动应用缺陷发现的 GUI Agent 评测基准

- **类型**：会议论文
- **状态**：NeurIPS 2026 在投，署名作者
- **导师**：桑基韬教授，北京交通大学
- **描述**：参与提出首个系统化评估 GUI Agent 自主缺陷发现能力的评测基准，覆盖 18 款真实开源 App、143 个 GitHub Issue 驱动的缺陷复现任务。本人主要参与设计 LLM-as-Judge 自动裁判框架，定义融合缺陷规约、截图与行为轨迹的多源判定方法及 Defect Recall/Precision/F1 指标体系，并完成 UI-TARS、GUI-Owl 等主流 Agent 的统一基准评测与对比分析。
- **关键词**：GUI Agent、LLM-as-Judge、缺陷检测、评测基准

## 基于多模态智能体协同推理的可解释视频取证框架

- **类型**：科研项目
- **资助**：北京市自然科学基金本科生"启研"计划，资助 5 万元
- **时间**：2026 年至今
- **导师**：赵耀教授（IEEE Fellow），依托科幻音视频智能处理北京市重点实验室
- **描述**：本人作为核心成员（团队共 2 人），负责设计基于 MCP 协议的异构工具链协同调度机制，研究 MoE 架构下物理规律、语义自洽与空时一致性三维度专家的正交化训练策略与跨模态证据融合方法，参与构建基于思维链诱导的结构化 n-元组可解释取证输出协议。
- **预期成果**：CCF-A/B 论文 1–2 篇
- **关键词**：多模态智能体、MCP 协议、MoE、视频取证、可解释 AI

## 基于 RISC-V 架构的 Rust 轻量级实时操作系统设计与实现

- **类型**：科研项目（大学生创新训练项目）
- **状态**：已通过结题，最终评级待公布（校级以上）
- **导师**：高勃教授，北京交通大学
- **描述**：基于 Rust 在 RISC-V 64 位架构上从零构建轻量级 RTOS 内核，实现抢占式多任务调度、基于原子指令的同步原语、8 项系统调用及 UART/VGA/PCI 等多设备驱动。本人负责编译器工具链适配与文件系统模块开发，通过将 unsafe 代码约束至最小边界，验证了 Rust 所有权机制在嵌入式实时系统中保障内存安全的工程可行性。
- **关键词**：Rust、RISC-V、RTOS、操作系统、内存安全
